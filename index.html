Project Domain

Detailed research background and methodology of the Smart Dairy AI System

📚 Literature Survey

Existing dairy monitoring systems focus on basic milk tracking but lack real-time AI-driven insights. Research highlights the use of lactation curve models, anomaly detection, and precision livestock farming techniques.

⚠️ Research Gap

Sri Lankan dairy farms lack intelligent, real-time monitoring systems. Most current solutions are manual, reactive, and do not utilize AI or predictive analytics.

❗ Research Problem

How can we design a smart system that accurately predicts milk production, detects anomalies early, and provides actionable insights for farmers in real time?

🎯 Research Objectives

  • Predict milk yield using AI models
  • Detect anomalies in real time
  • Improve dairy productivity
  • Provide farmer-friendly dashboards

🧪 Methodology

A hybrid approach combining Wood’s Lactation Curve with Random Forest models. Real-time data is processed using streaming pipelines and anomaly detection techniques.

🛠️ Technologies Used

  • Frontend: React / React Native
  • Backend: .NET / FastAPI
  • Database: SQL Server
  • AI/ML: Python, Scikit-learn
  • Deployment: Docker, AWS
